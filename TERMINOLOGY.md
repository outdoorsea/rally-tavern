# Rally Tavern Terminology

## Roles

| Term | Role | Example |
|------|------|---------|
| **Overseer** | Human who runs a Gas Town | You, me, any human collaborator |
| **Mayor** | AI orchestrator in a Town | Claude Code, Codex |
| **Polecat** | AI worker agent | Spawned by Mayor for tasks |
| **Deacon** | Background AI agent | Monitors, maintenance |

## Places

| Term | What |
|------|------|
| **Town** | A Gas Town installation (~/gt) |
| **Rig** | A project within a Town |
| **Rally Tavern** | This coordination hub |

## Work

| Term | What |
|------|------|
| **Bounty** | A task posted for anyone to claim |
| **Bead** | A work item in Gas Town/Beads |
| **Convoy** | A batch of related beads |
| **Round** | Distributing work to multiple agents |

## Communication

| Term | What |
|------|------|
| **Gossip** | Shared context/intel (has TTL) |
| **Mail** | Direct messages between parties |
| **Help** | Q&A between overseers |

## Knowledge

| Term | What |
|------|------|
| **Practices** | Best practices for codebases |
| **Starters** | Boilerplate templates |
| **Playbooks** | Step-by-step guides |
| **Learned** | Lessons from experience |
